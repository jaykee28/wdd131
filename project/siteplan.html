Site Purpose

The website provides an online presence for Wisdom and Word (PTY) Ltd, allowing clients to learn about and access programming lessons, printing, and other services easily.

Planned Pages and Content

  • Home: Introduction to the business and its mission.
  • Services: Computer lessons (HTML, CSS, C#, Java, Python), printing, photocopying, document scanning, stationery sales, internet café access.
  • Gallery: Examples of programming projects, class activities, and services.
  • Contact: Business location, phone number, and email.

Scenarios

  • How can I enroll in a programming lesson at Wisdom and Word?
  • What are the hours for printing, photocopying, and internet café services?

Color Schema

  • Dark Blue (#1E3A8A): Headings, navigation, primary accents.
  • Orange (#F97316): Buttons, links, highlights.
  • Light Gray (#F3F4F6): Background color.

Typography

  • Roboto: Headings (H1, H2, H3)
  • Arial: Body text
  • Courier New: Optional for code or project examples
